# maven-in-java
=======
## Simple Maven Hello World Project
### Requirements
- Java 11
- Maven 3

### How to run
- `git clone https://github.com/ubaid4j/Simple-Maven-Project.git`
- `cd Simple-Maven-Project`
- `mvn package`
- `java -jar target/*.jar`
### Expected Output
```text
Hello World
```
